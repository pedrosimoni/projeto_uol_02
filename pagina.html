Sobre Este Projeto

Este projeto é uma jornada prática para configurar um servidor Nginx na Amazon Web Services (AWS) e garantir que ele esteja sempre online. Ele me permitiu aplicar conhecimentos em:

  • **Infraestrutura em Nuvem (AWS):** Criando redes, máquinas virtuais e regras de segurança.
  • **Administração Linux:** Instalando e configurando serviços essenciais.
  • **Automação e Scripting:** Desenvolvendo scripts para monitorar a saúde do servidor.
  • **Monitoramento e Alertas:** Implementando notificações automáticas em caso de problemas.

A ideia é simular um ambiente real onde a disponibilidade de um serviço é crítica, e a equipe precisa ser alertada rapidamente se algo der errado.

O Que Foi Feito?

Em resumo, as principais etapas do projeto envolveram:

  • Montar uma rede personalizada na AWS (VPC).
  • Lançar um servidor Ubuntu na nuvem.
  • Instalar e configurar o Nginx para servir esta página.
  • Criar um "vigia" (script) que checa o site a cada minuto e registra tudo.
  • Programar alertas automáticos para o Discord se o site cair.
